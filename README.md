
# Auth Kata

A starter repository for the React authentication kata.

Read the kata brief before you start: [kata-react-auth.md](./kata-react-auth.md)

## Structure

```
Client/   - scaffolded React + Vite app (this is where you work)
Server/   - completed .NET auth API (provided, no changes needed, just configuration)
```

## Getting started

1. Fork this repository
2. Clone your fork
3. Follow the setup instructions in [Server/README.md](./Server/README.md) to get the backend running
4. Open `Client/` and start building
